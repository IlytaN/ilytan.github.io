Instagram

The Scenario

Nowadays, social media apps have taken up a great part of our life. Among them, Instagram has become the largest photo-sharing app and attract 1 billion users around the world. Instagram these days is a complex platform with many features being built on top of various technologies such as React Native, Django (Python), PostgreSQL etc.

Inspired by what Instagram achieved, I decided to develop a mini version of Instagram applying the technologies I had learned at my university instead of Instagram's real tech stack. The clone was planned to include some of Instagram's basic features which were User authorization, Post creation, Tagging, Searching and News feed (Like and comment). It was also important that the clone would provide a mobile-app experience and its UI must look the same with the real app.

The Product

A hybrid app was developed (the screenshot above) and it gave the users the same feeling as Instagram did. Users could install, sign up, sign in, create post with tags, like others' posts and search posts based on the tags. To do that, a client-server model was applied and two separate repositories were set up for the client application and the server. HTML, CSS, Ionic and AngularJS were made use of to create the mobile application. The server was created using NodeJS and it was deployed to heroku.com. Both the client app and the server were amazingly fast and secured when they communicated with each other using REST API

Technologies

AngularJS, NodeJS, Ionic, HTML, CSS

Pet Care System

The Scenario

Nowadays, owning a pet becomes more challenging for people due to the fact that pet health is affected by feeding schedule, while people cannot always stay at home to feed their dear friends. Although the urge to have an automatic pet feeder at home has raised significantly worldwide, it is not yet popular and affordable in many countries at the time. As a result, DIY pet care system can be an excellent choice for people who are not able to purchase automatic pet feeder.

The aim of this project was to create a pet feeding machine and a web application for remotely control the machine. It was also important to apply and study Raspberry Pi, ReactJS and NodeJS which are the most popular technologies from both IoT and Web development so that an objective evaluation could be given upon this approach.

The Product

A pet feeding machine and a web application were developed. The pet feeding machine included a food dispenser, a servo and Raspberry Pi. Pet’s food could be released from the food dispenser as a result of Raspberry Pi triggering the servo to rotate the dispenser’s spinner. NodeJS and Pigpio – a C library - were applied to receive the requests from the web app and manipulate the servo.

Although the web application's UI was simple, it was a mobile-friendly progressive web app with all the core features, such as Feed now and Schedule next feed included. ReactJS was used to make this application.

Medicool

The Scenario

Nowadays in Russia there is no useful application for searching medicines through the pharmacies. The customer needs to call the info desk and wait for about 5-7 minutes until the operator answers to get his medicine.

To solve such problems, this project was planned to include a website acting as an admin panel for pharmacy employees to manage medicine information, a mobile application for end-users to search medicines and pharmacies, a server and a shared database.

The Product

A 4-member team including me worked on this project from planning and designing phase to development and testing phase.

As a result, a website was made for the pharmacy employees to manage medicine information such as names, dosage, availability etc. and pharmacy information. All CRUD actions were available in this PHP web application. A database was also created with MySQLWorkbench and it worked perfectly with both this admin panel and the mobile application.

A mobile application was built using Ionic, AngularJS. Users could search medicine by name and location. They could also check the addresses of the nearby pharmacies which have the medicines they needed. This feature was developed using cordova map plugin and google map api.

A NodeJS server was created. It could work with the mobile application and the shared database.

Technologies

HTML, CSS, JavaScript, AngularJS, Ionic, jQuery, Bootstrap, NodeJS, MySQL, PHP, MySQLWorkbench
